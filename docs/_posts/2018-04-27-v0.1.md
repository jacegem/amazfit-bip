# Font Creator v0.1

Amazfit Bip 을 위한 폰트파일을 생성하는 프로그램입니다. 

![](https://goo.gl/ZwoBtY)


1. `Select TTF file` 버튼을 눌러서 폰트파일을 선택합니다. 

2. `Font File Create` 버튼을 눌러 폰트 파일을 생성합니다.

3. `ft 폴더`에 폰트가 생성됩니다.
